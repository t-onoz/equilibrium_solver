# Generic equilibrium solver for aqueous systems
## Supported
- acid-base equilibrium
- complexation
- precipitation of pure solids
- gas phases (Henry's law) \[open system only\]
- activity models
  - (extended) Debye-Huckel
  - Davies
  - Truesdell-Jones
 
## Not supported
- solid solutions
- closed system containing gas (e.g. soda can)
- two or more liquid phases (e.g. solvent extraction)
- redox half reactions (equilibrium containing e-)

## Usage
- please check jupyter notebooks in "examples" folder. `NaHCO3.ipynb` is relatively simple.
